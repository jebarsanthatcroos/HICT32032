# HICT32022 Week 4 - JWT Authentication API

## Project Description

This project is a simple Express.js REST API that implements JWT (JSON Web Token) authentication.

Users can:
- Login with username and password
- Receive a JWT token
- Access protected routes using the token

---

## Technologies Used

- Node.js
- Express.js
- JSON Web Token (JWT)
- JavaScript
